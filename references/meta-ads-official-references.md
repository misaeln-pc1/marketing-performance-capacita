# Meta Ads Official References

## Uso

Referencias oficiales para decisiones operativas de campanas Meta Ads. No reemplazan la configuracion vigente dentro de Ads Manager.

## Referencias V3 - Excel Presencial Santiago

- Traffic objective / landing page views: Meta documenta el objetivo Trafico para llevar personas a sitio web, app o evento, y senala que puede optimizarse para link clicks, landing page views o alcance unico diario.
  - https://www.facebook.com/business/ads/ad-objectives/traffic
- Reels ads / 9:16: Meta recomienda creatividad construida para Reels, incluyendo video vertical 9:16 con mensajes dentro de zona segura.
  - https://www.facebook.com/business/ads/facebook-instagram-reels-ads
- Advantage+ Creative / enhancements: Meta describe herramientas automaticas como generacion de texto, ajustes de imagen/video, expansion de imagen y otras variaciones creativas.
  - https://www.facebook.com/business/ads/meta-advantage-plus/creative
- Advantage+ Placements: Meta describe la automatizacion de placements en Facebook, Instagram, Messenger y Audience Network.
  - https://www.facebook.com/business/ads/meta-advantage-plus/placements
- Creacion de campanas en Meta Ads Manager / personalizacion por placement: Meta indica que Ads Manager permite previsualizar placements y que ciertas opciones dinamicas pueden impedir personalizar anuncios manualmente por placement a nivel de anuncio.
  - https://www.facebook.com/help/messenger-app/621956575422138/

## Lectura aplicada

Para la V3 de Excel presencial, estas referencias justifican:

- usar objetivo Trafico con optimizacion a visitas de pagina de destino;
- partir con video vertical 9:16 para el aprendizaje inicial movil;
- mantener desactivadas mejoras creativas automaticas cuando el objetivo sea aislar variables;
- mantener control manual de placements/creatividades cuando el aprendizaje requiere aislar formatos;
- no corregir una advertencia secundaria de columna derecha subiendo un asset aislado si no bloquea la publicacion y el test se concentra en Reels, Stories y Feed movil.
