# README Visual - Marketing Performance Capacita

## Proposito

Este mapa es una navegacion visual del proyecto Marketing Performance Capacita para Obsidian.

Debe abrirse desde un vault padre que contenga los repos Capacita como carpetas hermanas, por ejemplo:

```text
Capacita-GitHub/
├─ capacita-global-control/
└─ marketing-performance-capacita/
```

Desde ese vault, abrir:

```text
marketing-performance-capacita/visual/MAPA_MARKETING.canvas
```

## Fuente de verdad

Este mapa no reemplaza:

- `marketing-performance-capacita/PROJECT_CONTEXT.md`
- `marketing-performance-capacita/TASK_STATUS.md`
- `marketing-performance-capacita/REVIEW_REQUEST.md`

GitHub sigue siendo la fuente de verdad versionada. Obsidian es solo una capa visual de navegacion.

## Seguridad de datos

- Los datos reales de leads, clientes, metricas y presupuestos no deben subirse al repo.
- No incluir datos personales, secretos, tokens, credenciales ni exportaciones privadas.
- Los assets pesados deben ir en Google Drive o en la plataforma correspondiente.
- El repo solo debe contener documentacion operativa, estructura, criterios y referencias versionables.

## Alcance inicial

El mapa conecta:

- contexto y estado del proyecto;
- campana Excel Presencial Santiago;
- buyer personas;
- formulario Meta Ads;
- Zoho CRM;
- email, WhatsApp y seguimiento comercial;
- automatizacion actual y futura.
