# templates

Plantillas reutilizables para campañas, briefs, auditorías y documentación operativa.

Uso previsto:
- Briefs de campaña.
- Prompts operativos.
- Checklists.
- Plantillas de revisión.
- Estructuras reutilizables.

No incluir:
- Datos reales de clientes.
- Información personal.
- Credenciales.
