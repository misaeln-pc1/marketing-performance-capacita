# Esquema Universal de UTMs - Capacita.cl

## 1. Estructura de la URL
https://capacita.cl/curso?utm_source={CANAL}&utm_medium={FORMATO}&utm_campaign={PRODUCTO}-{MES}-{AÑO}&utm_content={PERFIL}

## 2. Diccionario de Valores para Mayo 2026
* utm_source: meta
* utm_medium: leadform
* utm_campaign: SCL-EXCEL-MAY26
* utm_content: desbordado o reinsercion

## 3. Logica de Negocio
Este esquema permite que Zoho CRM identifique automaticamente el perfil del lead para asignar el guion de ventas correcto (Productividad vs. Certificacion).
